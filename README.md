# Global-Workforce-Analytics-Dashboard-projet

## 📌 Project Overview

The **Global Workforce Analytics Dashboard** is a Power BI project developed to analyze employee data and workforce trends across different departments and countries. The dashboard provides interactive insights into employee demographics, salary distribution, company growth, and performance analysis to support data-driven HR decision-making.


## 🎯 Objectives

* Analyze workforce distribution across departments
* Understand gender diversity within the organization
* Study employee age distribution patterns
* Identify top earning countries
* Evaluate employee performance metrics
* Track company growth trends
* Enable dynamic employee filtering
* Analyze salary statistics department-wise


## 📊 Dashboard Features

### 👥 Workforce Analysis

* Number of employees in each department
* Gender distribution by department
* Employee filtering using the first letter of employee names
 
### 📈 Employee Demographics

* Age spread of employees using Histogram visualization
* Department-wise workforce insights

### 💰 Salary Analysis

* Top earning countries
* Minimum salary by department
* Maximum salary by department
* Average salary by department

### ⭐ Performance Analysis

* Performance spread analysis using sort-by-column functionality
* Employee performance comparison across departments

### 🚀 Organizational Insights

* Company growth trend analysis over time
* Workforce expansion monitoring
